# Zenith Dashboard
A minimalist, high-performance personal productivity dashboard for Chrome.

## Features
- **Live Widgets:** Real-time tracking for Formula 1, sports, weather, and more.
- **Customizable:** Drag-and-drop widget layout.
- **Privacy-First:** All configurations and shortcuts are stored locally in your browser.
- **Minimalist Design:** A clean, dark-mode aesthetic for a focused workspace.

## How to Install
1. Download the latest version of this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the folder.

## Privacy Policy
Zenith Dashboard does not collect, store, or transmit any user data. All data is managed locally within the browser.

## Built With
- Node.js & Express (API Backends)
- Chrome Extensions API
- ImageMagick (Icon Generation)
